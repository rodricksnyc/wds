What is this?

Overview

A Design System is the source which groups all the elements that will allow the teams to design, realize and develop a product. A good design system is:

  • Reusable
    A design system is a series of components that can be reused in different combinations. A Design System is not a deliverable, but a set of deliverables.
  • Manageable
    Design systems allow you to manage design at scale. A team can zero in on the components that they need most.
  • Ever-Evolving
    It will evolve constantly with the product, the tools and the new technologies. It can stand the test of time and grow to suit the needs of design and development teams.
